## 自定义组件

```
1、在外面创建一个目录
2、在此目录再次创建一个目录 然后在官方的开发工具中，右击，选择component菜单，名称我们一般和目录名一样
wxml js json wxss
3、调用，那里需要那里配置  json
{
   "usingComponents":{
       "标签名":"绝对地址，没有后缀  /components/share/share"
   }
}

在模板中  <自定义的标签名 />
```
