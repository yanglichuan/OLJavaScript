##Padding
```html
/* 上下左右都为10px */
padding: 10px; 
/* 上下10px  左右20px */
padding: 10px 20px;
/* 上10px 左右20px 下30px */
padding: 10px 20px 30px;
/* 上10 右20 下30 左40 顺时针*/
padding: 10px 20px 30px 40px;

width: 200px;
height: 200px;
border: 1px solid red;
padding: 20px;
margin:10px;
/* padding border margin 都会撑大 [带有宽高的]  盒子 */
```
