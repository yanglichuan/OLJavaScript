#Note
## 默认
h1默认字体大小是2em（2倍的基础字体大小）
Chrome浏览器默认字体大小是16px 那么h1就是32px大小

## 权重

| 类型     | 数值 |
| -------- | :--: |
| 标签     |  1   |
| 类       |  10  |
| ID       | 100  |
| 通配符   |  0   |
| 行内样式 | 1000 |
| !import  | 9999 |

## 文本样式属性
| 字体属性         |           说明          |
| -------------   | :----------------------: |
| word-spacing    |    设置元素内单词之间的距离  |
| letter-spacing  |    设置元素内字母之间的距离  |
| text-transform  |   设置元素内文本的大小写  |
| text-decoration |   设置元素内文本的装饰 |
